# nautical
nautical game engine


Controls:

Pause with 'P'

Move player with 'WASD'

Shoot grapple with mouse, release to retract

Zoom with ',' and '.'

Turn on darkness with 'O'

To switch between DEBUG_MODE, press enter

In DEBUG_MODE:

        Show bumpers with 'B'
